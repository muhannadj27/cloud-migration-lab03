# Cloud Migration and Cloud Adoption Lab

This repository presents a conceptual study of migrating a mid-sized retail company from an on-premises infrastructure to the cloud. The lab focuses on understanding PaaS, IaaS, and SaaS services across major cloud providers.

## Contents
- On-premises architecture design
- Cloud migration strategies
- Comparative study of cloud providers
- Architecture diagrams

This lab does not require hands-on cloud implementation and focuses on architectural and strategic understanding.
